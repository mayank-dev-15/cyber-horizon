# 🌃 Cyberpunk City

Interactive cyberpunk city with neon signs, flying cars & reactive audio.

## Features
- Procedural neon cityscape
- Animated flying cars
- Reactive audio visualization
- Cyberpunk aesthetic with glow effects

## Tech
- HTML5 Canvas
- Vanilla JavaScript
- Web Audio API

## Run
Open `index.html` in any modern browser.
